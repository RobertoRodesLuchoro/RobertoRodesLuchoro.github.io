ERP de Gestión de Taller - ProXpert Garage

Java Spring Boot JWT Dart Flutter

ProXpert Garage es una app para gestión de talleres mecánicos desarrollada como proyecto final de ciclo. Cuenta con backend en Java (Spring Boot), seguridad con JWT y Spring Security, y control por roles (mecánico, administrativo, jefe de taller y administrador). Permite gestionar órdenes de trabajo, balance financiero y una dashboard con métricas. Incluye API REST y frontend multiplataforma en Flutter.
